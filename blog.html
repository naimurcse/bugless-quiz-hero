The difference among map vs forEach vs filter vs find.

map

  • Purpose: "map" is used to transform each element of an array and create a new array with the same number of elements.
  • Behavior: It iterates over every element of the array, applies a callback function to each element, and collects the return values of the callback in a new array.
  • Return: It returns a new array. It doesn't modify the original array.

forEach

  • Purpose: "forEach" is used to iterate over an array and perform an action on each element without creating a new array.
  • Behavior: It iterates over the array, applies a callback function to each element, and does not return a new array. It is mainly used for its side effects, like logging or updating values.
  • Return: It doesn't returns a new array. It doesn't modify the original array.

filter

  • Purpose: "filter" is used to create a new array containing elements that meet a certain condition.
  • Behavior: It iterates over the array, applies a callback function to each element, and includes elements in the new array for which the callback returns true. It doesn't modify the original array.
  • Return: It returns a new array. It doesn't modify the original array.

find

  • Purpose: "find" is used to retrieve the first element in an array that meets a specified condition.
  • Behavior: It iterates over the array, applies a callback function to each element, and returns the first element for which the callback returns true. It stops iterating after finding the first match.
  • Return: It returns the first matching data. It doesn't modify the original array.
